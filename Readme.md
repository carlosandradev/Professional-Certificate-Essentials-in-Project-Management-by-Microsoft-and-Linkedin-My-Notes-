### 0. Introduction
---

**Deliver successful Projects**
* A project start when someone has a problem to colve or an opportunity to pursue.
* The job as a project manager is an ongoing balancing act between project scope and time, resources, and budget you've been given.

*Objectives of the course*
* Use a case of study to explore how to manage projects successfully.
* Look how to start a project the right way and then put together a comprehensive plan for running it.
* Lear how to keep your project on track from start to finish.

---

**What you should know**
* This course is only an introduction to project management.
* The basics to know what to expect as you work through the course.

*Course Topics*
* We start as the very begining: What is a project? What is project management?
* There are two examples of metodologies: Waterfall and Agile.

* Next two chapters are about building a project plan and schedule with the waterfall approach.

### 1. Get to know Project Management

**What is a Project**
* A Project is a temporary endeavor that has an unique goal and usually a budget.
---
* Temporary endeavor: A project has a definite beginning and end.
* If the project is implementing a new system, the project is done when you successfully hand it over to operations.
* What if the project seems to go on forever? it could be that you haven't clearly defined what you're trying to acomplish.
